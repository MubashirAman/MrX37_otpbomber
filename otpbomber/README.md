## Overview

Mrx37 OTPbomber is a tool designed to send multiple OTP (One-Time Password) messages to specified phone numbers, primarily for testing purposes. With enhanced features, it now supports customizable rounds of OTP sends with delays to manage rate limits. This tool is lightweight, user-friendly, and built for educational use.

## Installation

Follow these steps to download and set up the tool:


1. **Clone the repository**:
   ```bash
   git clone https://github.com/MubashirAman/MrX37_Otp_Bomber.git
   cd MrX37_Otp_Bomber
   ```

2. **Install required libraries**:
   Run the following command to install all dependencies.
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Once installed, you can use Mrx37 OTPbomber with various options to send OTPs to one or more phone numbers. Ensure phone numbers are in the correct format with the country code (e.g., +92 for Pakistan).
Basic Usage
Send OTPs to a single phone number:

# python3 core.py +923014565678

Multiple Rounds
Repeat OTP sends multiple times per number (e.g., 3 rounds):
# python3 core.py +923014565678 --counts 3

Single Website
Test a specific website (e.g., priceoye):
# python3 core.py +923014565678 --site priceoye
